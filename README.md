# Omnifood

Omnifood is a responsive food subscription landing page designed to showcase a personalized meal delivery service based on user tastes and nutritional needs.

This project focuses on **frontend fundamentals**, responsiveness, and performance optimization.

## Why This Project?

I built this project to strengthen my frontend skills, especially:
- Responsive design using breakpoints
- Image optimization for performance
- Writing clean and maintainable CSS
- Building layouts without frameworks

This project has **no backend yet** and focuses purely on the UI/UX layer.

---

## Features

- Fully responsive design (mobile, tablet, desktop)
- Optimized images for different screen sizes
- Clean and modern layout
- Interactive UI using JavaScript
